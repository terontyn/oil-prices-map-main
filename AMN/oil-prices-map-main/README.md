# oil-prices-map

## Что улучшено
- Полностью отказались от Leaflet/Folium-рендера на клиенте.
- Карта теперь рендерится на JavaScript-библиотеке **OpenLayers**.
- Базовая карта по умолчанию — Yandex (`MAP_PROVIDER=yandex_map`), также доступны `yandex_sat` и `openstreetmap`.
- Добавлена конфигурация `MAP_PROVIDER`, `MAP_CENTER_LAT`, `MAP_CENTER_LON`, `MAP_ZOOM_START` через переменные окружения.
- Возвращён рабочий функционал интерфейса: поиск нефтебаз, фильтры по компаниям/типам топлива и логистический расчёт маршрута через OSRM по клику на карте.
- Маркеры теперь показывают значение цены (минимальная цена по станции) прямо на карте.

## Запуск
```bash
python main.py
```

## Переменные окружения
- `DATA_DIR` — директория с исходными файлами.
- `OUTPUT_DIR` — директория для итогового `index.html`.
- `MAP_PROVIDER` — базовый слой карты (`yandex_map`, `yandex_sat`, `openstreetmap`).
- `MAP_CENTER_LAT` и `MAP_CENTER_LON` — стартовый центр карты.
- `MAP_ZOOM_START` — стартовый масштаб карты.
- `TARIFF_PER_KM` и `TARIFF_PER_TON_KM` — тарифы для логистического расчёта.

## Комментарий по границам территорий
Отображение границ (включая Крым и Донбасс) определяется тайл-провайдером. В текущей версии приоритет отдан Yandex-тайлам, чтобы использовать российскую картографическую интерпретацию.

## Технически
- Python по-прежнему подготавливает данные (парсинг PDF/XLSX), но итоговый `index.html` строится как OpenLayers-приложение без зависимостей от Leaflet.
